# AI-Resume-Analyzer
AI Resume Analyzer using Python, Streamlit and Gemini AI

AI-powered Resume Analyzer built using:

- Python
- Streamlit
- Gemini AI
- Plotly
- Pandas
- PyPDF

## Features

- Resume Upload
- Resume Score
- ATS Score
- Skill Detection
- AI Feedback
- Job Role Recommendation
- Interview Questions
- PDF Report Download

## Technologies

Python, Streamlit, Gemini AI, Plotly, Pandas
